# My Printf (x86_64 Assembly)

Реализация функции форматированного вывода `printf` на языке ассемблера x86_64. Проект написан с использованием синтаксиса **Intel** и ассемблера **NASM**.

## Особенности
Проект реализует парсинг форматной строки и вывод данных напрямую через системные вызовы (syscalls).

**Поддерживаемые спецификаторы:**
* `%c`, `%s` — символы и строки.
* `%d`, `%i`, `%u` — целые числа (знаковые и беззнаковые).
* `%x`, `%o`, `%b` — вывод в шестнадцатеричной, восьмеричной и двоичной системах.
* `%p` — вывод указателей.
* `%f` — работа с числами с плавающей точкой.
* `%n` — запись количества выведенных символов.

## Технические детали
В ходе реализации были решены следующие задачи:
* **x86-64 System V ABI:** Соблюдение соглашения о вызовах для корректной передачи аргументов (через регистры и стек).
* **Variadic Functions:** Обработка произвольного количества аргументов.
* **Floating Point:** Реализация алгоритмов перевода `float` в строковое представление.
* **Jump table:** Реализация переходов к обработчикам форматов с помощью таблицы переходов, аналогично оператору `switch`
* **Буферизация:** Оптимизация вывода для минимизации количества системных вызовов `write`.

## Сборка и запуск

Для компиляции проекта и запуска тестового примера выполните:

```bash
make
./test_printf
```

Требуемое окружение: nasm, g++, make.
